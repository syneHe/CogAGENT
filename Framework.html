Framework

System design and architecture

Multimodal, knowledgeable and controllable conversational frameworks

Multimodal agent implements Perception, Response, Enhancement and Embodied functions for modeling multimodal dialogue context. Knowledgeable agent integrates Structured Knowledge, Unstructured Knowledge and Implicit Knowledge to enhance dialogue generation. Controllable agent constructs Empathic Reply and Dialogue Safety to make response generation under control.

Comprehensive conversational models, metrics and datasets.

CogAGENT implements 17 conversational models covering task-oriented dialogue systems, open-domain dialogue systems and question answering systems. Moreover, it provides 8 conversational metrics including ROUGE, BLEU and so on. Meanwhile, our toolkit integrates 11 classical conversational datasets. All of them can be called one-click.

Open-sourced and modularized conversational toolkits.

CogAGENT is proposed as an open source toolkit. Hence, users can modify codes for their own customised models or datasets. It implements comprehensive conversational agents in a modularized interface. Our toolkits decouple codes and present easy-to-use modules named Reader, Processor, Model, Metric and Trainer to provide a low barrier entry for developers and researchers.

Oline dialogue system.

We have also released a playful online conversational system, which support conversational agents interacting with users.

Enhancer module details of CogKTR

BaseModel class is the base class of all models in CogKGE. BaseModel class organizes code into three basic sections: (1) forward function for training, (2) embedding function for getting the embedding of entities and relations, (3) scoring function for computing the score of triples. Model module consists of four parts: translation distance models, semantic matching models, graph neural network-based models and transformer-based models.
